# NGINX（Ubuntu）

1. 查看版本 
```shell
nginx -v
```

2. 安装
```shell
sudo apt update
sudo apt install nginx
```

3. 启动
```shell
nginx -c /etc/nginx/nginx.conf 启动
nginx -s reload 重启
nginx -s stop 停止
```

4. systemctl 命令

	- 启动Nginx
```shell
sudo systemctl start nginx 启动
```

	- 重启Nginx
```shell
sudo systemctl reload nginx 重启
```

	- 停止Nginx
```shell
sudo systemctl stop nginx 停止
```

	- 查看 Nginx 运行状态
```shell
sudo systemctl status nginx 状态
sudo systemctl enable nginx 开机自启动
```

5. 编辑*.conf
```shell
nano /etc/nginx/*.conf
```


6. 查看生效的nginx配置
```shell
nginx -T/-t
```

7. 查看nginx启动情况
```shell
ps -ef|grep nginx
```

8. 杀进程
```shell
kill -9 pid
```
