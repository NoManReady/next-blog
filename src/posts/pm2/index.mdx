# PM2

## 进入项目根目录

- 启动进程/应用

```shell
pm2 start bin/www 或 pm2 start app.js
```

- 重命名进程/应用

```shell
pm2 start app.js --name www
```

- 添加进程/应用

```javascript
pm2 start bin/www --watch
```

(--watch：监听应用目录的变化，一旦发生变化，自动重启。如果要精确监听、不监听的目录，最好通过配置文件)

- 结束进程/应用

```javascript
pm2 stop www
```

- 结束所有进程/应用

```javascript
pm2 stop all
```

- 删除进程/应用

```javascript
pm2 delete www
```

- 删除所有进程/应用

```javascript
pm2 delete all
```

- 列出所有进程/应用

```javascript
pm2 list
```

- 查看某个进程/应用具体情况

```javascript
pm2 describe www
```

- 查看进程/应用的资源消耗情况

```javascript
pm2 monit
```

- 查看 pm2 的日志

```javascript
pm2 logs
```

- 若要查看某个进程/应用的日志,使用

```javascript
pm2 logs www
```

- 重新启动进程/应用

```javascript
pm2 restart www
```

- 重新启动所有进程/应用

```javascript
pm2 restart all
```
